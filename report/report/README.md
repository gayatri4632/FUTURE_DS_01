# 📊 Business Sales Performance Analysis – Detailed Report

## 🎯 Objective
To analyze business sales data and identify trends in products, categories, and regions.

---

## 📁 Dataset
Superstore Sales Dataset

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Column)

---

## 📊 Analysis Performed

### Category Analysis
- Sales and profit comparison across categories
- Identified best performing category

### Top Products Analysis
- Found highest revenue generating products

### Region Analysis
- Compared sales across different regions

---

## 🔍 Key Insights

- Technology category performs best in revenue
- A few products generate most of the sales
- Sales vary significantly across regions

---

## 📈 Conclusion
This project demonstrates how data analysis helps businesses make better decisions using Excel-based insights.

Time-based analysis was considered but not included due to dataset structure.
