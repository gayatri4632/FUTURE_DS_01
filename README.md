# FUTURE_DS_01
Business Sales Performance Analysis – Task 1

## 🎯 Objective
This project analyzes business sales data to understand revenue trends, top-performing products, and category-wise performance.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs

---

## 📊 Key Analysis
- Sales and profit trends
- Top-performing products
- Category-wise performance
- Regional sales distribution

---

## 🔍 Insights
- Certain products generate the majority of revenue
- Some categories outperform others significantly
- Sales vary across regions

---

## 💡 Conclusion
This analysis helps businesses understand sales performance and make data-driven decisions.
